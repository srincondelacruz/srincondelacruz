## 🚀 Proyectos destacados

### 🩺 [NoShow Predictor](https://github.com/srincondelacruz) · *TFM (en desarrollo)*
Modelo de predicción de ausencias para una clínica médica con datos reales anonimizados. **~95% accuracy** con XGBoost/LightGBM y explicabilidad mediante SHAP.

`Python` · `XGBoost` · `LightGBM` · `SHAP`

### 📄 [DeathClause](https://github.com/srincondelacruz/DeathClausule)
SaaS full-stack para análisis de contratos: extrae y vectoriza cláusulas de PDFs, detecta contradicciones internas con búsqueda semántica + GPT-4o y genera resúmenes ejecutivos con chat RAG.

`Python` · `FastAPI` · `React` · `Azure OpenAI` · `ChromaDB` · `D3.js`

### 💀 [Startup Death Oracle](https://github.com/srincondelacruz/startup-death-oracle)
App RAG que predice modos de fallo de startups. Pipeline Medallion (Bronze→Silver→Gold) sobre 409 casos reales, búsqueda semántica con ChromaDB y generación con Gemini 2.0 Flash.

`Python` · `Databricks` · `ChromaDB` · `Gemini API` · `Streamlit`

### 🧬 [Activation Steering](https://github.com/srincondelacruz/activation-steering)
*Activation steering* sobre GPT-2 con TransformerLens, abliteration en Qwen2.5-0.5B-Instruct y un editor de personalidad en Gradio con visualización 3D en Three.js.

`PyTorch` · `TransformerLens` · `Gradio` · `Three.js`

### 🍴 [theSpoon](https://github.com/srincondelacruz/theSpoon)
Previsión de demanda para restaurantes con variables de clima, estacionalidad y festivos. Pipeline ML completo de extremo a extremo.

`Python` · `scikit-learn` · `Pandas`
